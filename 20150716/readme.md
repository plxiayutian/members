##分析一下JS执行流程  
##控制器
- 一个控制器控制一小块视图
- 不要试图在控制器里操作DOM 要用指令
- 不要进行数据格式化 要用过滤器
- 不要互相调用控制器 通过事件来实现
##模型
绑定在$scope
默认是一个空对象，你可以往里面赋值
- $scope 默认是一个空对象
- $scope提供了一些方法 $watch $apply
- $scope 是上下文环境
- $scope是一个树型结构，与DOM标签平行
- 每一个angular应用都有一个根作用域 $rootScope
- $scope可以传播事件 可以向上也可以向下
